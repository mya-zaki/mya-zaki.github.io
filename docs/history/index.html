2016/09 〜 2017/04 : 新規 CtoC Sharing Economy サービス立ち上げ

サービスの構成はスマートフォンのアプリ、Web（API, 管理ツールなど）、バッチ処理となっていて、
体制はパートナーの方を含めた開発チームと一部を外注ですすめていたプロジェクト。
プロジェクト途中の参加でしたが、方針転換があったこともあり見直しや作り直しが多く発生するタイミングで、
外注の方とは主に仕様のやりとりなどのコミュニケーションを担当。
API 設計の見直し、業務の確認を行いタスクを洗い出し、他パートナーの方のタスク管理についても担当。
